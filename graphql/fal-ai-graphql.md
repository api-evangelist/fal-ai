# fal GraphQL API

fal is a fast serverless inference platform for AI models including image generation (Stable Diffusion, FLUX, Kling), video generation, speech, and custom models. The API covers model invocation, queue management, and webhook results.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from fal REST/gRPC API
**Documentation:** https://fal.ai/docs

- Reference: https://fal.ai/docs/model-apis/quickstart
- Schema: https://github.com/api-evangelist/fal-ai
